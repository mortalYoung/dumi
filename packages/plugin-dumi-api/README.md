# plugin-dumi-api

[![NPM downloads](http://img.shields.io/npm/dm/plugin-dumi-api.svg?style=flat)](https://npmjs.org/package/plugin-dumi-api)



## Install

```bash
# or yarn
$ npm install
```

```bash
$ npm run build --watch
$ npm run start
```

## Usage

Configure in `.umirc.js`,

```js
export default {
  plugins: [
    ['plugin-dumi-api'],
  ],
}
```
